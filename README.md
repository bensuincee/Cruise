# Crusié – Gezi ve Etkinlik Otomasyon Sistemi

Crusié, kullanıcıların şehir, tarih ve kategori tercihlerine göre etkinlikleri görüntüleyebildiği ve bilet işlemlerini yönetebildiği masaüstü tabanlı bir gezi ve etkinlik otomasyon sistemidir.

## Project Overview
Bu proje, kullanıcıların etkinlik arama, bilet alma, iptal etme ve favorilere ekleme işlemlerini gerçekleştirebildiği; görevli kullanıcıların ise etkinliklerde görev almasını sağlayan çok modüllü bir sistem olarak geliştirilmiştir. Projede nesne yönelimli programlama (OOP) prensipleri ve dosya tabanlı veri yönetimi kullanılmıştır.

## Features
- Kullanıcı kayıt ve giriş sistemi
- Şehir, tarih ve kategoriye göre etkinlik arama
- Etkinlik seçimi ve detay görüntüleme
- Bilet satın alma ve iptal işlemleri
- Kullanıcıya ait biletlerin listelenmesi
- Favori etkinlik ekleme ve silme
- Görevli kullanıcılar için etkinliklerde görev alma sistemi
- Dosya tabanlı veri saklama ve güncelleme

## Technical Details
- GUI geliştirme (Java Swing)
- Dosya tabanlı veri yönetimi (TXT dosyaları)
- HashMap ve ArrayList kullanımı
- ActionListener ile kullanıcı etkileşimleri
- OOP prensipleri: Inheritance, Polymorphism, Encapsulation, Abstraction

## Technologies Used
- Java
- Java Swing (GUI)
- Object-Oriented Programming (OOP)
- File I/O
